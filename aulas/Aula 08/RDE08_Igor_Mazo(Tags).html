Tags

  • <table>: essa tag serve para criar tabelas, utilizando a tag <tr> para definir uma linha e <td> para definir uma coluna, e para incluir linhas na tabela, basta colocar "border="numero" dentro da tag table (<table border="1">)
    • Ex:
    Nome Idade Profissão
    Juca 32 Médico
    Leopoldo 29 Veterinário
  • <del>: essa tag serve para riscar alguma palavra ou frase no arquivo
    • Ex:A tag del faz isso.
  • <ins>: essa tag serve para destacar alguma parte do texto, sublinhando.
    • Ex:A tag ins faz isso.
  • <sub>: essa tag faz com que a palavra fique em um tamanho menor e abaixo da linha.
    • Ex:A tag sub faz isso.
  • <sup>:essa tag faz com que a palavra fique em um tamanho menor e acima da linha.
    • Ex:A tag sup faz isso.
  • <blockquote>: essa tag serve para indicar que o texto escrito é uma citação longa
    • Ex:
    Essa é uma definição tirada da Mozilla Developer Center.
  • <q>: essa tag serve para indicar que o texto é uma citação curta
    • Ex:De acordo com o site mdn web docs: O elemento HTML <q> indica que o texto dentro da tag é uma pequena citação.
  • <abbr>: essa tag é usada para fazer uma abreviação.
    • Ex:W3S é um ótimo site para aprender programação.
  • <adress>: essa tag é usada para informar meios de contato ou endereços.
    • Ex: numero de telefone: 99999-9999
  • <cite>: essa tag serve para definir um título de algum trabalho.
    • Ex:Monalisa de Leonardo da Vinci.
  • <bdo>: essa tag serve para indicar qual é a direção do texto, usando dir=rtl ou dir=ltr dentro da tag
    • Ex:Esse texto vai da direita para a esquerda
  • <code>essa tag serve para formatar o texto, fazendo-o parecer com um texto antigo
    • Ex:Esse texto está com uma fonte antiga.
  • <pre>: essa tag faz com que o texto já venha pré-formatado exatamente como está no programa.
    • Ex:
    Este texto
        está
            pré formatado
  • <kbd>: essa tag serve para formatar um texto como entrada de teclado.
    • Ex:Aperte as teclas Ctrl + C para copiar um texto.
